# 🎓 University Management System (Java OOP Project)

## 📌 Description

This is a Java-based mini project developed to demonstrate core **Object-Oriented Programming (OOP)** concepts. The project models a simple university system and focuses on key principles such as **inheritance, polymorphism, method overriding, and access control**.

---

## 🎯 Objectives

* Demonstrate the use of **inheritance** in Java
* Show **polymorphism** through method overriding
* Explain the use of **`this` and `super` keywords**
* Illustrate **final classes and methods**
* Practice **clean and structured Java programming**

---

## 🧠 Concepts Covered

* Inheritance (`extends` keyword)
* Superclass vs Subclass
* Single Inheritance
* Multilevel Inheritance
* Hierarchical Inheritance
* Polymorphism (Runtime)
* Method Overriding
* `this` Keyword
* `super` Keyword (methods & constructors)
* `super()` Constructor Calls
* Protected Access Modifier
* Packages and Visibility
* Final Class
* Final Method
* Method Hiding (Static Methods)
* Implicit `super()` Rules

---

## 🏗️ Project Structure

```
university/
 ├── Person.java
 ├── Student.java
 ├── GraduateStudent.java
 ├── Teacher.java
 ├── Admin.java
 ├── Utility.java
 └── MainApp.java
```

---

## ⚙️ How It Works

* `Person` is the **base class** containing common attributes like name and age
* `Student` and `Teacher` inherit from `Person` (**hierarchical inheritance**)
* `GraduateStudent` extends `Student` (**multilevel inheritance**)
* Method overriding is used to customize behavior in subclasses
* Polymorphism allows a `Person` reference to refer to different object types

---

## ▶️ How to Run

### 1. Compile

```bash

```

### 2. Run

```bash
java university.MainApp
```
